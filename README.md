# Tentsile Setup Calculator

**[Live app](https://rangoy.github.io/tentsile/)**

A small web app for pitching a Tentsile 3-point tree tent. Enter the trees in
your grove and the distances between them, and it tells you which 3-tree
combinations will work, ranks them by how much margin they leave, and
calculates the strap length needed for each corner. Includes a top-down
visualization of the trees, tent, and straps.

Not affiliated with or endorsed by [Tentsile](https://www.tentsile.com/).

See [SPEC.md](SPEC.md) for the full geometric model and design decisions.
